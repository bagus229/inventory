# Judul: Sistem Manajemen Inventaris Barang (E-Inventory) Berbasis Vue.js dan CodeIgniter 4.

## Deskripsi Singkat: 
E-Inventory merupakan sistem informasi berbasis web yang dirancang untuk membantu proses pengelolaan persediaan barang secara terintegrasi. Sistem ini menyediakan fitur manajemen data kategori, supplier, stok, barang, serta pencatatan histori transaksi keluar dan masuk barang. Aplikasi ini dibangun menggunakan Vue.js sebagai frontend dengan konsep Single Page Application (SPA) dan CodeIgniter 4 sebagai backend yang menyediakan layanan REST API. Dengan begitu pengelolaan akan lebih efektif dan terintegrasi.
